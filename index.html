🍎 macOS 首次打开

Cortex 暂未购买苹果开发者签名（¥99/年），首次启动会弹"无法验证开发者"。放行方法：

  1. 双击 Cortex.app → 弹警告 → 点"完成"
  2. 系统设置 → 隐私与安全性 → 底部"仍要打开"
  3. 或终端跑：
xattr -dr com.apple.quarantine /Applications/Cortex.app

🪟 Windows 首次打开

Windows SmartScreen 会拦截未签名应用（同上：未买代码签名证书）。放行方法：

  1. 双击 Cortex-Setup.exe
  2. SmartScreen 弹"未识别的应用" → 点"更多信息"
  3. 下方出现按钮"仍要运行" → 点它

部分杀软可能误报，请加白名单后再装。

常见问题

FAQ

Cortex 会拿到我的 API key 吗？ ⌄

不会。API key 用 AES-256 加密存在你电脑的 ~/.cortex/data.db，加密主密钥单独放本机。Cortex 没有任何远端服务接收你的密钥。源码里 secrets_vault 模块可查证。

Cortex 推荐我做什么策略？ ⌄

Cortex 是工具不是顾问。内置网格 / 趋势跟踪 / 均值回归 / DCA 等经典策略，参数你定。AI 助手能基于市况给建议，但最终决策由你。所有策略都强制建议你先在模拟盘跑一周。
