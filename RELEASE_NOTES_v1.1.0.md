## 🎯 主要功能

### 5 种策略全面升级
- **现货网格**：等差 + 等比两种模式，alt 币种用等比覆盖更均匀
- **智能 DCA**：定投基础上加买入条件（RSI 超卖 / 价格低于 SMA20 / 双过滤），避开高位
- **双均线**：SMA / EMA / WMA 三选，支持做多 / 做空 / 双向
- **马丁加仓**：完整支持合约做空（逢涨加仓 + 跌止盈镜像逻辑）
- **AI 自主交易**：5 大 LLM 平台 + 全套指标 + 订单簿 + 资金费率，长短双向 6 路决策

### 专业级回测
- 8 大业界标配指标：总收益 / Sharpe / Sortino / Calmar / Profit Factor / Recovery Factor / 最大回撤（含持续时间）/ 胜率
- 水下回撤曲线（Underwater Curve），看清最深亏多少、亏多久
- AI 自主交易也能回测（指标规则近似业绩基线）
- 参数自动调优（多目标排序，一键回填）
- 中文参数名（不再有 `fast_period` / `lower_price` 之类的英文）

### 风险保护
- 启动前参数预检：网格远离市价 / 马丁深度超账户 / 模拟盘不允许跑合约 — 全部前置拦截
- 长短双向的紧急平仓：多头自动 SELL / 空头自动 BUY 平仓，不会反向加仓
- AI 接口连续失败时：先按持仓方向发市价平仓单 → 再停策略 → critical 级通知（不留裸奔仓位）
- 智能 DCA 历史不足时跳过买入，不退化成"无脑定投"

### 体验
- 首次安装可选主题（4 套：深灰蓝 / 暖色 / 纯黑 / 浅色）
- 收益仪表盘 + 30 天累计 PnL 曲线
- 多账户多策略统一管理
- 完整中文界面（所有参数标签、错误提示）
